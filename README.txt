This is the locator app for android.

This app is not published in google play.
In order to run it you need to do the following steps:

1. Run the locator server application (instructions are in the location server application).
2. Download android studio, and clone this project.
3. Add the google api key to local.properties. The google api key is located in my drive, near the other passwords.
4. Run the app in android studio.

################################################################################

This app contains 3 screens:

1. login screen - appears in the first installation of the app.
2. home screen - contains a search bar to search for other available users.
3. map screen - after choosing a user, it shows its last known location.

Note:
For future development, see the Current_status file (in this project) that contains
the current status of the app.
